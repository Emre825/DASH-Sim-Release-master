◆ run_sim_initial_dataset()

def DTPM_utils.run_sim_initial_dataset (   sim_params)

Call the simulator with specific parameters in order to generate the dataset that will be used for the DTPM policies.

This method is called multiple times in parallel, based on the availability of CPU cores in the host machine. Each simulation executes and obtains the results from a single snippet (one sample in the dataset).

Parameters
sim_paramsSimulation configurations that change among different simulations, for instance, DVFS config and snippet.
